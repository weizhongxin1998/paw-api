# Paw API — UI 设计

## 1. 组件库

| 项 | 选择 | 备注 |
|---|---|---|
| UI 框架 | **Naive UI** | Vue 3 + TypeScript 原生支持，tree-shaking，主题系统灵活 |
| 图标 | **Ionicons v5**（`@vicons/ionicons5`） | 线条极简风格，与 Naive UI 默认搭配 |

## 3. 主题系统

### 3.1 白天模式

- **默认主色**：Naive UI 原生绿色（#18a058）
- **可切换主题色**：内置 3-4 套预设（绿、蓝、紫等），通过 Naive UI `useTheme` / `NConfigProvider` 切换
- 切换范围：主色、辅色、选中态等品牌色变量

### 3.2 黑夜模式

- 统一深色主题，**不切颜色主题**
- 使用 Naive UI 内置 `darkTheme`
- 通过配置式切换，Phase 1 即搭好基础设施

### 3.3 切换机制

- 标题栏提供主题切换入口
- 白天模式内部进一步选择颜色主题（配置面板）
- 状态持久化到 localStorage + 用户设置存储

## 4. 组件选型（Naive UI 映射）

| 界面区域 | Naive UI 组件 |
|---|---|
| 标题栏 | `NLayoutHeader` / `NSpace` |
| 集合树 | `NTree` + 自定义渲染 |
| Tab 栏 | `NTabs`（`type="card"`）+ 拖拽排序（第三方或自实现） |
| Method 下拉 | `NSelect` |
| URL 输入 | `NInput` |
| 发送按钮 | `NButton`（color=primary） |
| 请求选项标签 | `NTabs`（Params/Headers/Body/Auth） |
| 键值对编辑 | `NDynamicInput` 或自封装 |
| Body 编辑 | `NInput`（textarea）+ 可选择格式 |
| 响应状态 | `NTag` |
| 响应体 | `NInput`（readonly, monospace）或代码高亮 |
| 环境选择器 | `NSelect` / `NDropdown` |

## 5. 设计原则

- 保持一致：所有颜色、间距、字体遵循 Naive UI 设计令牌
- 桌面优先：布局固定宽度，不响应式
- 信息密度适中：参考 Postman/Apifox 的间距比例
